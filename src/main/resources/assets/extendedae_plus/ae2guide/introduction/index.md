# Extendedae_plus 指南

欢迎使用 **Extendedae_plus**。这是一个基于 Applied Energistics 2 (AE2) 的附属模组，在 ExtendedAE 的基础上进行了功能增强，聚焦于「样板供应器」的改进与玩家体验优化，并可与多种 AE2 扩展生态（如 AAE等）共存。 它提供更灵活、可控的样板执行行为，以及更直观、友好的 GUI 交互。
